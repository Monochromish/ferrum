# Ferrum

## Commands

### `npm run start`
Start Electron + dev server with HMR, using Nollup.

### `npm run dev:ui:rollup`
Start dev server with HMR, using Rollup.

### `npm run dev:ui:livereload`
Start dev server with livereload.

### `npm run build:ui`
Build UI

### `npm run serve:ui`
Serve UI

### `npm run lint`
Lint code
